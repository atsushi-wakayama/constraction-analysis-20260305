🌱 政務活動費 管理ツール（ベータ版）
==================================

【かんたん起動 ✨】
このフォルダの中で、お使いのPCに合わせて1つだけクリックしてください:

  📍 Macを使っている方     →  「🌱 アプリを起動 (Mac).command」をダブルクリック
  📍 Windowsを使っている方 →  「🌱 アプリを起動 (Windows).bat」をダブルクリック

ブラウザが自動で開いてログイン画面が出ます。
パスワードは初期値 「beta1234」 です。

----------------------------------------

【「📁 アプリ本体（さわらないでください）」って何？】
中身はプログラム本体とあなたのデータです。
さわらなくて大丈夫です。フォルダごと安全な場所にバックアップ
していただければ、データの引っ越し・復元ができます。

----------------------------------------

【🆘 動かないとき】

▶ Mac:「開発元が未確認です」と表示される
   → 「🌱 アプリを起動 (Mac).command」を 右クリック → 「開く」 → 「開く」
     を選んで一度許可してください。次回からは普通にダブルクリックでOK。

▶ どちらのOS:「Node.js がインストールされていません」と表示される
   → https://nodejs.org/ja を開いて「LTS」と書かれた緑のボタンを
     クリック → ダウンロードしたものをインストール → もう一度ランチャー
     をダブルクリック。
     ※ 一度インストールすれば次回以降は何もしなくてOKです。

▶ 領収書のPDFアップロード機能を使いたい
   → 別途 poppler のインストールが必要です（任意）。
     Mac:     ターミナルで  brew install poppler
     Windows: 領収書機能は使わないなら不要です。

----------------------------------------

【パスワードを変更したい】
1) 「📁 アプリ本体（さわらないでください）」 フォルダを開く
2) 「.env.local」というファイルをメモ帳などで開く
3) ADMIN_PASSWORD=beta1234 の右側を好きな文字に書き換えて保存
4) もう一度ランチャーで起動

----------------------------------------

【データの保存場所】
すべてのデータは「📁 アプリ本体（さわらないでください）」内の
「data/」 フォルダに保存されます:
  - payments.json     支払証明書
  - activities.json   活動記録簿
  - ledger.json       個人会計帳簿
  - vehicle-trips.json 自家用自動車使用簿
  - receipts.json + receipt-images/  領収書

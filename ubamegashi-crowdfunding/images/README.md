# 画像配置ガイド

このフォルダに以下のファイル名で写真を置いてください（JPG/PNG どちらでもOK、推奨は JPG）。

| ファイル名 | 使用箇所 | 推奨サイズ |
|---|---|---|
| `hero-kiln.jpg` | ヒーロー背景（職人＋窯の炎の写真） | 1920×1080 程度、横長 |
| `landscape-saplings.jpg` | STORYセクション右側（耕作放棄地に植えられた苗木と山並み） | 1200×1500 程度、縦長または正方形 |
| `sapling-kiln.jpg` | 「苗木から、炭へ。」セクション背景（苗木と窯の炎） | 1920×1080 程度、横長 |

## ファイルサイズの目安

そのままだと写真が重くて表示が遅いので、できれば以下を推奨：

- 1枚あたり 200〜500KB
- 横幅 1600〜1920px 以内
- JPEGクオリティ 80〜85%

オンラインの圧縮ツール例：
- https://squoosh.app/ （Google製・ブラウザ完結）
- https://tinypng.com/

## 写真がない場合

ファイルがなくても、CSSのフォールバック背景色（深緑グラデーション）でサイトは表示されます。
